# 关于本站

本站旨在收集各类资料用作学习分享使用，包括但不限于各科期中期末考试题、各种教辅电子书、优秀笔记，让信息差带来的差距少一点、再少一点，让每个考试人都能无障碍获取所需，为你的考试保驾护航。

所有资源均来自与公开网络。分享只为研究交流学习，提升技术使用，您下载的资源不可进行任何形式的贩卖与传播，你必须在 24h 内删除所有资源。

让我们一起支持正版，共同维护有序的商业环境，下载资源就视为你同意上述条款！

# 使用指南：

分类模块对应不同科目、不同类别的资料，可自行食用。所有链接均为直链，点击即下。所无所需书籍还请在对应模块下留言资料名称，每周均会进行更新。

# 留言系统：

在对应文章下留言需要的资料文件名称，如果有同学有分享链接的亦可通过留言系统进行回复，及时高效。同时管理员也会每周日更近网站内的文件资料，各位可以常来看看～

# 打赏：

1. 通过捐款链接的打赏：该打赏将用于【西电科大书籍资料群：627029256】群主 @白桦进行筹款，具体筹款规则见【筹款模块】。
